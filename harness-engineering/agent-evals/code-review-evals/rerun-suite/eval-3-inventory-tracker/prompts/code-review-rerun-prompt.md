# Code Review Agent — Inventory Tracker Rerun

## Context

Review the inventory tracker implementation in `project/src/` against the spec
in `project-brief.md` and the Programmer handoff in `reports/programmer-handoff.md`.

## Deliverables

1. Read the spec, implementation, tests, and Programmer handoff.
2. Produce a structured code review covering:
   - correctness and spec alignment
   - boundary conditions and aggregate invariants
   - code quality and error handling
   - test quality and missing boundary tests
   - security and audit integrity
3. Classify findings by severity.
4. Save the review to `reports/code-review-report.md`.
