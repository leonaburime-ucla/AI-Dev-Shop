# Continuity Reports

Use this folder for non-feature `progress-ledger.md` files when work is resumable but not attached to a single pipeline feature folder.

Example:

```text
reports/continuity/
  harness-tripwires-rollout/
    progress-ledger.md
```

Feature-bound work should keep its ledger in `reports/pipeline/<NNN>-<feature-name>/progress-ledger.md` instead.
