# Workflows

This folder is part of the committed `project-knowledge/` template.

Use the writable `ADS-project-knowledge/meta/workflows/` workspace for project-owned operating workflows that:

- come from a retained debate, report, or experiment
- should guide future work in one project
- are not yet adopted as framework-wide workflow rules

Promote a workflow from `ADS-project-knowledge/meta/workflows/` into `framework/workflows/` only after an explicit decision to make it part of the toolkit itself.
