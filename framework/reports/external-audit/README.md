# External Audit Reports

Use this retained folder only when the user explicitly wants to keep an external-audit artifact as reusable project evidence.

Default local-only paths:

```text
.local-artifacts/external-audit/
  packets/
  runs/
  offloads/
```

Retained layout:

```text
framework/reports/external-audit/
  packets/
  runs/
```

Packets summarize the work handed to the external auditor. Run reports capture:

- what work was audited
- what the external auditor said
- what the Coordinator agreed with
- what should change
- what the Coordinator disagreed with and why
